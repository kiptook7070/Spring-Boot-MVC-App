# Spring-Boot-MVC-App

Create a Spring Boot starter project using any IDE like STS, Netbeans, Intellij Idea etc.
While creating Starter Project select ‘Spring Web’, ‘Spring Data JPA’, ‘MySQL Driver’, ‘ Thymeleaf’, ‘Lombok’ and ‘Spring Boot DevTools’ as 
starter project dependencies. 
Here ‘Lombok’ and ‘Spring Boot Dev Tools’ are optional to add.
